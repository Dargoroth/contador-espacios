Consejo: si el navegador hace scroll al presionar Espacio, haz clic en cualquier parte de la página primero, o usa el botón Iniciar para activar el contador.

Cómo usarlo

  1. Haz clic en Iniciar.
  2. Presiona la tecla Espacio tantas veces como quieras.
  3. Usa Pausar para detener el tiempo sin perder el conteo.
  4. Reiniciar pone todo a cero.

Acceso rápido: también puedes presionar Enter para Iniciar/Pausar y R para Reiniciar.
